# scripts

Utilitários de desenvolvimento (a partir da Fase 1):

- `run-local.sh` — sobe a stack local (compose + profiles)
- `seed-assets.sh` — popula assets/usuários de laboratório
- `test-connectivity.sh` — valida isolamento de rede (usuário NÃO alcança o
  asset; gateway alcança) e allowlist de portas
- geração de certificado autoassinado para o Nginx local
